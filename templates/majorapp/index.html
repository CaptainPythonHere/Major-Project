About

How Our Journey Of PHISHING DETECTION Began.

  • 2020-2021

    Problem Statement

    Phishing is a type of cyber security attack in which attackers clone a website and get access to sensitive information about the user.

  • October 2020

    Our motive

    A lightning bolt idea struck in our heads to create such a solution that detects phishing URLs and save people from this social engineering attack.

  • March 2021

    Our Approach

    Understanding how the detection of such websites is important, we have come up with a Supervised MACHINE LEARNING Model that detects the PHISHING Websites. The User can check the website before entering it whether it is phishing or not.

  • July 2021

    Additional Functionality to the Browser

    It can be used as an additional Fuctionality to the browser. Due to which it will be easier for the user to access this web-Application
